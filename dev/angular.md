# Dependencies

- [Node.js *Node.js runtime*](/dev/nodejs)
- [VS Code *IDE*](/dev/vscode)

## New Project
- Create a new project
```
ng new <project-name>
```
- Enter the folder
```
cd <project-name>
```
- Start Dev Server at `http://localhost:4200/`
```
ng serve --open
```
- Open in VS Code
```
code .
```
